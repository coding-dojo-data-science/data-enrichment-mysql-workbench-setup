(Windows)MySQL Server Installation

Installation (Windows 10)

  1. Download the required MySQL software from the MySQL website Choose the MySQL Installer MSI Option.

You can skip signing up for an account 

Once the installer has downloaded, open it and agree to let it make changes to your computer.

Then Select “Custom”

Open the “MySQL Servers” menu

From the menu, choose MySQL 8.0 and then the top version. Add it to the "Products to Be Installed" box with the arrow

Then open the “Applications” Menu and select the top most version of MySQL Workbench. Add it to the "Products to Be Installed" box with the arrow.

Once you have selected both the MySQL Server and MySQL workbench, click next to continue.

 

2. Follow the installer guidelines, with the following considerations:

    • Choose the password "root" for your root user password
    • When asked about which authentication method you should pick, DO NOT select the first method but select 'Use Legacy Authentication Method' (see below).

For "Windows Service" make sure you have the "Configure MySQL Server as a Windows Service" and "Start the MySQL Server at System Startup" options selected.

The installer will configure your Windows system to run the MySQL Server automatically on startup, listening on the default port 3306, as well as establish the required environment variables to run MySQL tools from a Windows shell.

Open a MySQL shell from CMD/PowerShell or GitBash to make sure everything is working correctly:  (You can find this by typing CMD into the tool bar search and opening the Command Prompt App)

mysql -u root -p and enter "root" when prompted for the password.

Success will look like this:

Troubleshooting:

Problem: Running mysql in CMD/PowerShell gives the following error: 'mysql' is not recognized...

Solution: Make sure you have an Environment Variable PATH to your MySQL executables. Likely to be in C:\\Program Files\MySQL\MySQL Server 5.7\bin

If you encounter any problems during installation, please do not hesitate to consult your instructor.
