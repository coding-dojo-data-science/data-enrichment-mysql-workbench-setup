(Mac)MySQL Server Installation

Installation (Mac)

Installation Steps:

  1. Use this link MySQL Download and select the macOS and DMG option.
  2. Open the download when finished and follow the steps.  Be sure to choose the Legacy Password Encryption option.
  3. Create a password and remember it, it will be used to connect to the MySQL server later.

Accessing the MySQL server:

  1. Open the System Preferences pane and select the MySQL server.
  2. This will be where we will start/stop the MySQL server.

Create mysql alias for terminal connection:

Run the below command in the terminal window

echo $SHELL
  • If the response ends with zsh, then use this command in the terminal window
echo 'alias mysql="/usr/local/mysql/bin/mysql"' >> ~/.zshrc && source ~/.zshrc
  • If the response ends with bash, then use this command in the terminal window
echo 'alias mysql="/usr/local/mysql/bin/mysql"' >> ~/.bash_profile && source ~/.bash_profile


Let's take a look at what line is doing.

  • echo will write to a file.
  • alias mysql="/usr/local/mysql/bin/mysql"
    • alias will create a variable for our terminal to use called mysql.
    • /usr/local/mysql/bin/mysql is the location of the mysql executable we just installed.
  • >> will append to a file.
  • ~/.zshrc (  is a file that the terminal window will look at for any commands to run.
  • && will run another command.
  • source ~/.zshrc (or source ~/.bash_profile) will make the alias we just created available in the terminal window.

Connect to mysql shell via terminal:

Run the below command in the terminal window.

mysql -uroot -p

Enter your mysql password when prompted.  If you see this, you have correctly connected to the mysql server via the terminal.  If not see your instructor for help.  Type quit to exit the shell.
